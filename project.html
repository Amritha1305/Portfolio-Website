SkyVerse - Weather App


SkyVerse is a simple and interactive weather application that allows users to check real-time weather conditions for any city worldwide. Built using HTML, CSS, and JavaScript, it fetches live weather data from an API and displays temperature, weather conditions, and more in an intuitive interface.

Features:

  • Search weather by city name

  • Real-time weather updates

  • Clean and responsive UI

Tech Stack: HTML, CSS, JavaScript, Weather API

What Next - To-Do List App


What Next is a simple and intuitive to-do list application designed to help users stay organized and manage tasks efficiently. Built with HTML, CSS, and JavaScript, the app allows users to add, mark as complete, and delete tasks. It also uses local storage to save tasks, ensuring they persist even after the page is refreshed.

Features:

  • Add and remove tasks easily

  • Mark tasks as completed

  • Tasks persist using local storage

  • Clean and user-friendly interface


Tech Stack: HTML, CSS, JavaScript, Local Storage

Digital Clock


This Arduino-powered digital clock is designed to display real-time hours, minutes, and seconds with high accuracy. Using an RTC (Real-Time Clock) module, the clock maintains precise time even when power is lost. It features a seven-segment display / LCD for clear time visibility and can be enhanced with features like alarm functionality, temperature display, or LED indicators.

Features:

  • Real-time clock with high accuracy

  • Display time in HH:MM:SS format

  • Battery backup for continuous operation

  • Expandable with alarm, temperature, or LED effects

Tech Stack: Arduino, RTC Module, LCD/7-Segment Display, C++ (Arduino IDE)
